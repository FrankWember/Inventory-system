# BarTrack - Inventory Management System

A professional inventory management system for bars and restaurants, built with React Native and Expo. Track stock, manage sales sessions, monitor trends, and analyze financial performance.

## Features

- **Inventory Management**: Track stock levels with smart alerts for low stock items
- **Session Management**: Record daily purchases, conduct inventory counts, and calculate sales
- **Financial Tracking**: Monitor revenue, profit margins, and expenses
- **Trends & Analytics**: View performance metrics and top-selling items
- **Multi-Unit Support**: Handle both individual units and cassiers (boxes) for beers

## Tech Stack

- **Framework**: React Native with Expo
- **Database**: Supabase
- **Navigation**: React Navigation
- **Language**: TypeScript
- **Platforms**: iOS, Android, and Web

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Expo CLI

### Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env` file with Supabase credentials
4. Start: `npm start`

## Deployment

Configured for web deployment using Expo static export. Deploy to Vercel with one click.
